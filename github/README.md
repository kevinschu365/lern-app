# LernApp

Statische Lern-App fuer deine Module mit Quizfragen, offenen Fragen, Uebungen und Lerntipps.

## Lokal starten

`app-ready.html` im Browser oeffnen.

## Fuer GitHub Pages

1. Repository auf GitHub anlegen.
2. Alle Dateien aus diesem Ordner hochladen.
3. `Settings` > `Pages` oeffnen.
4. Als Source den Branch `main` und den Root-Ordner `/` waehlen.
5. Danach ist die App ueber GitHub Pages erreichbar.

## Wichtige Dateien

- `app-ready.html`: saubere App-Startseite
- `main.js`: saubere App-Logik fuer GitHub / Browser
- `real-modules.js`: Inhalte der Module
- `styles.css`: Layout und Design
